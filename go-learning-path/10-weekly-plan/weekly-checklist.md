# 每周目标 + 验收 checklist（8 周）

## 第 1 周
目标：掌握 Go 语法基础并完成 CLI
- [ ] 能写结构体与方法
- [ ] 能用切片/map 完成统计
- [ ] CLI 支持参数与错误提示
- [ ] README 描述用法与示例

## 第 2 周
目标：掌握模块与错误处理
- [ ] `go mod` 能正确依赖外部包
- [ ] 统一错误返回策略
- [ ] 待办清单支持文件持久化

## 第 3 周
目标：掌握 HTTP 与 JSON
- [ ] `/todos` 支持 GET/POST/DELETE
- [ ] JSON 编码/解码无 panic
- [ ] 接口返回统一 JSON 结构

## 第 4 周
目标：具备可测试性
- [ ] 至少 5 个表驱动测试
- [ ] 覆盖边界场景（空列表、非法输入）
- [ ] `go test ./...` 全绿

## 第 5 周
目标：掌握并发模式
- [ ] worker pool 可控并发数
- [ ] 任务汇总与错误处理完善
- [ ] 给出性能对比或日志

## 第 6 周
目标：接入数据库
- [ ] 使用 SQL 迁移或初始化脚本
- [ ] 数据增删查改完整
- [ ] 错误处理不泄露内部细节

## 第 7 周
目标：形成完整 API 服务
- [ ] Blog API 完整 CRUD
- [ ] 分页/排序/搜索可用
- [ ] README 写明 API 规范

## 第 8 周
目标：可部署
- [ ] Dockerfile 可一键启动
- [ ] OpenAPI/接口文档完善
- [ ] 部署步骤可复现
